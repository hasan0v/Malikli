## 5. Workflows

**User Registration and Login:**

1.  User navigates to the registration page.
2.  User enters their personal information (e.g., name, email, password).
3.  User clicks the "Register" button.
4.  The system validates the user's information and creates a new user account.
5.  The user is redirected to the login page.
6.  User enters their email and password.
7.  User clicks the "Login" button.
8.  The system authenticates the user and redirects them to their account dashboard.

**Product Browsing and Purchase:**

1.  User navigates to the product catalog page.
2.  User browses the available products or uses the search function to find specific items.
3.  User clicks on a product to view its details.
4.  User adds the product to their shopping cart.
5.  User navigates to the shopping cart page.
6.  User reviews the items in their cart and adjusts quantities if necessary.
7.  User proceeds to the checkout page.
8.  User enters their shipping and billing information.
9.  User selects a payment method and enters their payment details.
10. User reviews their order and clicks the "Place Order" button.
11. The system processes the payment and creates a new order.
12. User receives an order confirmation email.

**Product Drop Workflow (Admin):**

1.  Administrator/Product Manager logs into the admin panel.
2.  Administrator/Product Manager navigates to the product drop management section.
3.  Administrator/Product Manager creates a new product drop.
4.  Administrator/Product Manager adds products to the drop, specifying quantities and pricing.
5.  Administrator/Product Manager sets the start date and time for the drop.
6.  Administrator/Product Manager saves the product drop.
7.  The system automatically releases the products at the scheduled time.
